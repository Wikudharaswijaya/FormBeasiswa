# FormBeasiswa
Aplikasi ini digunakan untuk pendaftaran beasiswa antar negara.
Dalam Form ini user harus mengisikan :
  - Nama
  - Hobi
  - Ingin ke negara mana
  - Jenis Kelamin

Validasi Form :
  Semua Field harus di isi
